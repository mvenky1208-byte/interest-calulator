# Simple Interest Calculator

A Bash script to calculate simple interest based on user input.

## Description
This project contains a Bash script that calculates simple interest using the formula:

**Simple Interest = (Principal × Rate × Time) / 100**

The script prompts the user to enter the principal amount, rate of interest, and time period, then computes and displays the simple interest.

## Formula
Simple Interest = (P × R × T) / 100

Where:
- **P** = Principal Amount (the initial sum of money)
- **R** = Rate of Interest (percentage per year)
- **T** = Time Period (in years)
